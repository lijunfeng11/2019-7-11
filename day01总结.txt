13 阐述标识符的组成规则和注意事项
	标识符 : 给项目,包,类,方法,变量...起名字的符号
	标识符的组成规则 : a-z  A-Z  0-9  汉字(不推荐)  _  $ 
	
	起名子规范 : 了解!
		A 小驼峰  : 变量 , 方法
 *  		一个单词全部小写 , 两个单词以上从第二个单词开始首字母需要大写
 *  		int number = 10; 		int maxNumber = 20;		
 *  	B 大驼峰 : 类 , 接口
 *  		每个单词首字母都需要大写
 *  		HelloWorld	
 * 		C 包 
 * 			域名倒写, 每个单词全部小写
 * 			com.baidu
 * 
 * 		D 自定义常量
 * 			每个单词全部大写, 并且多个单词之间用下划线分割开
 * 			NUMBER   MAX_NUMBER
	
	注意事项 : 
		1 起名字不能是java中关键字
		2 起名字不能以数字开头

14 独立编写测试变量定义和使用的案例
	
	public class Demo1{
		public static void main(String[] args){
			// byte
			byte b = 10;
			System.out.println(b);
			
			System.out.println("-------");
			
			// short 
			short s = 20;
			System.out.println(s);
			
			System.out.println("-------");
			
			// int 
			int i = 30;
			System.out.println(i);
			
			System.out.println("-------");
			
			// long 
			long l = 1000L;
			System.out.println(l);
			
			System.out.println("-------");
			
			//float 
			float f = 12.12F;
			System.out.println(f);
			
			System.out.println("-------");
			
			// double 
			double d = 13.13;
			System.out.println(d);
			
			System.out.println("-------");
			
			// char
			char c = 'a';
			System.out.println(c);
			
			System.out.println("-------");
			
			// boolean
			boolean bb = true;
			System.out.println(bb);
			
			System.out.println("-------");
		
		}
	}

16 独立编写测试隐式类型转换的案例

	类型转换
		1 隐式转换(自动类型提升)
			不能类型数据作运算会产生类型提升 
			类型提升的规则 : byte,short,char->int->long->float->double
		2 强制转换
			把取值范围大的数据赋值给小的范围的类型才会涉及到强制转换
			强转的格式 : 目标类型 变量 = (目标类型)(被转换的数据);
				缺点 : 1 数据丢失  2 损失精读
			
			byte b1 = 10;
			int i = 20;
			
			byte b2 = (byte)(b1 + i);
			
			byte b3 = (byte)i;
